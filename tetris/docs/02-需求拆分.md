## 一、任务总览

Tetris 项目按 8 个阶段实施，每个阶段包含若干子任务：

| 阶段 | 内容 | 工作量 | 状态 |
| --- | --- | --- | --- |
| 0 脚手架 | Vite + TS + Tailwind + shadcn | 0.5d | ✅ |
| 1 配置 + 7 种方块 + SRS | CONFIG + tetrominoes + srs | 0.5d | ✅ |
| 2 核心算法 Board | 碰撞 + 锁定 + 消行 + Ghost | 1d | ✅ |
| 3 GameEngine | 状态机 + 7-bag + Hold + 计分 + 等级 | 1d | ✅ |
| 4 Canvas 渲染器 | 多区域 + 动画 + 消行闪烁 | 0.5d | ✅ |
| 5 状态层 Zustand | 桥接 UI 与引擎 | 0.3d | ✅ |
| 6 UI 层 React | HUD + 菜单 + 暂停 + 结束 + 设置 | 0.5d | ✅ |
| 7 单元测试 | Board / Tetromino / Bag / GameEngine | 0.5d | ✅ |
| 8 文档 + 验证 | 6 篇文档 + README + 部署 | 0.5d | ✅ |
| **合计** | | **5d** | ✅ |

## 三、按阶段的详细任务

### 阶段 0：脚手架

- [x] 创建 `games/tetris/` 目录
- [x] 初始化 Vite + React + TypeScript 项目
- [x] 配置 Tailwind + PostCSS + shadcn/ui
- [x] 设置 path alias `@/*`
- [x] 配置 ESLint + Prettier
- [x] 复制 `lib/utils.ts`、`lib/storage.ts` 自 2048
- [x] 复制 `components/ui/*` 自 2048（button / card / badge / dialog / switch / separator / label）
- [x] 暗黑紫调 `index.css` + `tailwind.config.ts`
- [x] `vercel.json` SPA 重写规则

### 阶段 1：配置 + 方块定义

- [x] `src/config/index.ts`：CONFIG 全量常量 + Zod schema + 类型导出
- [x] 7 种方块的 4×4 形状矩阵（tetrominoes.ts）
- [x] 旋转算法（rotateCW / rotateCCW / rotate180）
- [x] 4 旋转状态缓存（getRotations）
- [x] SRS 踢墙表（srs.ts）：KICKS_JLSTZ + KICKS_I + reverse + getKicksCW/CCW

### 阶段 2：Board 核心算法

- [x] `Board` 类：22 行（含 2 行缓冲区）
- [x] `isValidPosition`：碰撞检测
- [x] `lockPiece`：锁定到网格
- [x] `findFullLines`：找出满行
- [x] `clearLines`：消除 + 上方下沉
- [x] `getGhostY`：硬降位置
- [x] `getHardDropDistance`
- [x] `isGameOver`：出生位置被占用
- [x] `isColumnOverflow`：缓冲区溢出
- [x] 21 个单元测试（含 reset、边界、消行、Ghost、Game Over）

### 阶段 3：GameEngine 编排

- [x] GamePhase：`menu` / `playing` / `paused` / `over`
- [x] 60Hz 固定逻辑步长主循环（LOGIC_STEP = 1000/60）
- [x] 重力下落（按当前等级 tickRate）
- [x] 软降（1/10 速度）
- [x] 硬降（一键到底 + 计分）
- [x] 旋转：CW / CCW / 180°
- [x] 7-bag 随机生成（Bag.take / peek / next / reset）
- [x] Next 队列（预填 4 个 = 3 显示 + 1 缓冲）
- [x] Hold 系统（含 holdUsed 锁）
- [x] 消行检测 + NES 计分 + 连击
- [x] 等级提升（每 10 行 + 1，速度按 0.85 系数递减）
- [x] Game Over 判定 + 破纪录持久化
- [x] 21 个 GameEngine 单元测试

### 阶段 4：Canvas 渲染器

- [x] 多区域布局：Hold + 主网格 + Next
- [x] 已锁定方块 + 当前方块 + Ghost

- [x] 方块绘制：圆角 + 左上高光 + 右下阴影
- [x] Ghost 绘制：半透明轮廓
- [x] 消行闪烁动画（3 次 / 400ms）
